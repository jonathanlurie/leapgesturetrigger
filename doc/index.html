_runHalfRule(halfRule, hand)

[PRIVATE] runs a half-rule on a given hand. A half-rule is a single element of a rule (that is an array of size max 2). In other words, a rule can be composed of 2 half-rule if it applied to 2 hands. Some rules apply only to a single hand, implicitely they are already half-rules encapsulated in an array

fingerExtendedCode(hand)

for a given hand, get a code based on what fingers are extended. Each extended finger add up to a code to create a unique checksum

  • thumb + 1
  • index + 2
  • middle + 4
  • ring + 8
  • pinky + 16

Parameters

  • Hand hand :

    a leap js hand object

Returns

Number :

the code
